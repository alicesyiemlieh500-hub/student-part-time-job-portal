# student-part-time-job-portal

The Part-Time Job Board is a web-based platform designed to connect students with part-time job opportunities both on and off campus.
 The system allows students to easily search and apply for available jobs, while employers can post job openings and manage applications efficiently. 
Technologies Used: 
 PHP 
MySQL 
HTML 
CSS 
JavaScript 
 Bootstrap Target Users: 
 Students: Search and apply for part-time jobs. 
 Employers: Post job listings and review applications
 Git (for cloning the repository).
